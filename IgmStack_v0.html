Есть HTML код с последовательностью изображений (архив прикреплён). Есть фреймворк ImageStackV0.js, который вставляет скрол для этих рисунков.
Необходимо написать код на чистом Javascript или переделать существующий ImageStackV0.js так, чтобы:
- после i секунд рисунки скролились автоматически и зацикленно по j рисунков;
- по клику на рисунок - пауза и показать имя файла;
- по следующему клику - рестарт скролла.
i и j должны браться из инпутов в HTML коде.
